# Attendance Taker App

An Android application for tracking and managing attendance records efficiently.

## Project Structure

- **app/**: Main application source code
- **gradle/**: Gradle wrapper configuration
- **build.gradle.kts**: Project-level build configuration (Kotlin DSL)
- **settings.gradle.kts**: Gradle settings configuration

## Technology Stack

- **Platform**: Android
- **Language**: Kotlin/Java
- **Build System**: Gradle with Kotlin DSL
- **IDE**: Android Studio (based on .idea directory)

## Getting Started

### Prerequisites

- Android Studio Arctic Fox or later
- JDK 11 or higher
- Android SDK with minimum API level support

### Installation

1. Clone the repository:
git clone https://github.com/pranav6266/attendance-taker-app.git


2. Open the project in Android Studio

3. Sync Gradle files

4. Run the app on an emulator or physical device

## Building the Project

### Using Android Studio
1. Open the project in Android Studio
2. Click "Run" or press Shift + F10

## Linux/Mac
./gradlew assembleDebug

## Windows
gradlew.bat assembleDebug
